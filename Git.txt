git init //Se utiliza para declarar que se usara Git en el proyecto

git add *.* OR *carpeta o archivo* //Se utiliza para agregar al area de
espera por el Commit

git commit //Se utiliza para poner en estado de espera 
para subirlo ya al Repositorio proyecto o archivo

git status //Muestra que archivos no han sido *add* y tambien muestra los que 
tienen cambios despues de un *add*

git checkout -- *nombre con ruta del archivo* //Sirve para retroceder algun 
cambio que se haya hecho

git diff *nombre con ruta del archivo* //Muestra las diferencias de un cambio 
con el anterior

git log //Se usa para ver los commits que se han hecho

git commit -m "Mensaje del commit" //Es otra manera de hacer commit de manera 
mas rapida

git branch //Muestra las otras versiones del proyecto

git branch *nombre de la nueva version* //Crea una nueva version del proyecto

git checkout *nombre de la version a la que me quiero mover* //Cambia a la 
version indicada del proyecto

git clone *link del repositorio u proyecto a clonar* //Clona cualquier proyecto 
subido a un repositorio

git remote add origin *Link del repositorio* //Se establece cual sera el 
repositorio al cual se subira el proyecto

git push -u origin master //Sirve para pushear ya el proyecto al repositorio

Se puede crear un archivo en el proyecto el cual debe ser llamado 
.gitignore, En el se escribe el nombreruta de la carpeta o archivo el cual se
 quiere ignorar, Despues se puede subir comun y corriente

git push origin <your_branch_name> --force (Se usa despues de cambiar el remote)

rm -f ./.git/index.lock //sirve para eliminar el archivo index.lock que no permite hacer commit

git pull origin master --allow-unrelated-histories

git checkout -b new_branch old_branch

You could also disable use of the Git credential cache using 
"git config --global --unset credential.helper". Then reset this, and you would continue to have the cached credentials available for other repositories (if any). You may also need to do 
"git config --system --unset credential"(Use in CMD with Permisos de superusuario).helper if this has been set in the system configuration file (for example, Git for Windows 2).
